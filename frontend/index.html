¿Qué es el picking? Es separar la mercadería del depósito en bultos para cada cliente. Esta pantalla te muestra todo lo que hay que juntar en el día.

¿Cómo busco un artículo? Hay tres formas:
— Escaneás el código de barras con el botón Escanear (apuntá la cámara al código del producto)
— Escribís el código de barras a mano en la cajita y tocás Buscar
— Escribís parte del nombre del artículo en "Buscar por descripción..."

¿Qué hago con cada tarjeta? Cada tarjeta es un artículo para un cliente. Arriba dice el nombre del cliente, abajo el artículo. Usá los botones − y + para registrar cuántas unidades separaste, o tocá el número del medio para escribirlo directo.

¿Qué significa el estado?
— Pendiente: todavía no separaste nada
— Entregado parcial (tarjeta naranja): separaste menos de lo que pedía el cliente
— Completado: separaste todo lo que pidió el cliente

Botón "Solo pendientes": filtra la lista para mostrar únicamente los artículos que todavía no fueron tocados. Muy útil cuando ya van varios y querés ver lo que falta.

"Recientes": los últimos artículos que buscaste aparecen como chips para volver a encontrarlos rápido sin tener que escribir de nuevo.

La barra de arriba muestra el % del pick total completado entre todos los operarios.

Recientes:

¿Para qué sirve esta pestaña? Muestra el resumen del pick agrupado por cliente. En lugar de ver artículo por artículo, acá ves cuánto lleva separado cada cliente de todo su pedido.

¿Qué muestra cada fila? El nombre del cliente, su zona, el importe total del pedido, y una barrita de progreso que indica cuánto del pedido ya está separado.

Los filtros de arriba:
— Todos: muestra todos los clientes de la semana
— Completo: solo los clientes donde ya está todo separado
— Incompleto: los clientes donde se separó algo pero no todo (entrega parcial)
— Pendiente: los clientes donde todavía no se separó nada

¿Para qué toco una fila? Al tocar un cliente se abre un desglose con todos los artículos de ese cliente y cuánto se separó de cada uno. Útil para revisar antes de cerrar el bulto.

El buscador filtra por nombre de cliente. Si no te acordás bien cómo se escribe, con escribir dos o tres letras ya aparece.

Los clientes están ordenados por importe, de mayor a menor. Los más importantes aparecen primero.

¿Para qué sirve esta pestaña? Para registrar los artículos que sobran al final del picking. Escaneás o buscás cada producto, ingresás cuántas unidades o bultos quedaron, y exportás la lista para mandársela a administración.

¿Cómo se usa?
1. Elegí una lista existente o creá una nueva con + Nueva
2. Escaneá el código de barras del producto sobrante o escribilo a mano
3. Ajustá las unidades y los bultos con los botones − y +
4. Cuando terminés, tocá Exportar Excel para descargar la lista

Si escaneás un producto que ya está en la lista, la app lo resalta en lugar de duplicarlo.

Las listas no se comparten entre Yaguar y DIARCO — cada mayorista tiene las suyas.

Acá se gestionan todos los clientes. Cada cliente tiene un código Yaguar único, un nombre, una zona y un vendedor asignado.

Usá + Nuevo para registrar un cliente nuevo — el sistema asigna automáticamente el próximo código disponible. Tocá cualquier fila para editar los datos de un cliente existente.

Si modificás el nombre o la zona de un cliente, todos los picks de ese cliente se actualizan automáticamente.

Si aparece una sección Sin registrar arriba de la tabla, son clientes que tienen picks importados pero todavía no tienen datos cargados.

Semanas cargadas
Importar nueva semana

Exportá el archivo desde la app Yaguar: Menú → Sincronizar → Exportar base de datos. Cada vendedor tiene su propio archivo.

Podés subir varios archivos a la vez — uno por vendedor. El sistema los procesa todos juntos y arma el pick completo de la semana.

Si reimportás una semana que ya existe con el mismo nombre, se reemplazan todos sus picks anteriores.

Importar nueva semana

Los importes que muestra la app incluyen IVA y se calculan proporcionalmente sobre los artículos que se pickean.

Se excluyen automáticamente los ítems de sistema que no se pickean físicamente, como heladeras exhibidoras y artículos sin código numérico.

Si reimportás una semana que ya existe con el mismo nombre, se reemplazan todos sus picks anteriores.

Exportá el archivo desde la app DIARCO: Menú → Backup → MobileAssistantBU.db.

Podés subir varios archivos si tenés backups de distintos vendedores — el sistema los combina automáticamente.

Cada persona que usa la app necesita su propio usuario. Hay tres roles:

Operario — solo ve los picks de su turno, no tiene acceso al panel Admin.

Admin — puede importar semanas, gestionar clientes y exportar Excel, pero no puede cambiar roles de otros usuarios.

Superadmin — acceso total, incluyendo cambiar roles. Solo puede haber uno (el usuario ADMIN).

Para eliminar un usuario tocá el ✕ en su fila. No se puede eliminar el superadmin ni el único usuario que quede.

Las zonas son las localidades de los clientes (MERLO, VILLA LARCA, etc.). Cada zona debe estar asignada a un reparto para que los picks aparezcan en el orden correcto al pickear.

Los repartos agrupan zonas que salen en el mismo vehículo (Sur Abajo, Sur Arriba, Merlo, etc.). El orden de los repartos determina cómo se organizan los picks — usá las flechas ▲▼ para cambiarlo.

Cuando se importa una semana, las zonas nuevas que aparecen se crean automáticamente. Asignales un reparto acá para que queden ordenadas.
